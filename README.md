## Requirements

Ruby 2.2.0 +
Rails 4.2.0 +

##Installation

	$ git clone git://github.com/Gaolz/rails101s
	$ cd rails101s

##Run

	$ bundle install
	$ rake db:migrate
	$ rails s

##Testing
	$ rake test
